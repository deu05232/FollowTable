# Typical 'Great Firewall' Targets — Domains Without Exact URLs

| Type | Service | Domain | Common User Action |
| --- | --- | --- | --- |
| Search | Google | google.com | Open homepage and search |
| Video | YouTube | youtube.com | Watch a video |
| Social | Facebook | facebook.com | Log in / view feed |
| Social | Instagram | instagram.com | Browse posts |
| Community | Reddit | reddit.com | Read threads |
| Messaging | WhatsApp | whatsapp.com | Send messages |
| Messaging | Signal | signal.org | Register / message |
| Productivity | Google Translate | translate.google.com | Translate text |
| Cloud | OneDrive (occasionally impacted) | onedrive.live.com | Sync files |
| Developer | GitLab.com | gitlab.com | View repositories |
