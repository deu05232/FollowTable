# Windows Alt Codes (Alt+Number) for Common Currency Symbols

| Alt code (hold Alt + type) | Symbol produced | Common name |
|---:|:---:|---|
| Alt+0128 | € | Euro sign |
| Alt+0162 | ¢ | Cent sign |
| Alt+0163 | £ | Pound sterling sign |
| Alt+0165 | ¥ | Yen sign |
| Alt+0835 | ₣ | Franc sign |
| Alt+1423 | ₧ | Peseta sign |
| Alt+158 | ₧ | Peseta sign (alternate) |
| Alt+3647 | ₺ | Turkish lira sign |
| Alt+8377 | ₩ | Won sign |
| Alt+8355 | ₳ | Austral sign |
