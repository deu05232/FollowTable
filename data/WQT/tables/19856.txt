# Adobe Photoshop Compatibility Matrix by Operating System

| OS version | Supported Photoshop release | Notes |
| --- | --- | --- |
| Windows 7 (SP1) | Photoshop CC 2018 | Requires SSE2-capable CPU |
| Windows 8.1 | Photoshop CC 2019 | GPU acceleration recommended |
| Windows 10 (1607+) | Photoshop CC 2020 | Supports HiDPI scaling |
| Windows 10 (21H2) | Photoshop 2022 | Windows Defender exclusions suggested for scratch disks |
| Windows 11 (22H2) | Photoshop 2023 | WDDM 2.x driver required |
| macOS 10.13 (High Sierra) | Photoshop CC 2018 | Last supported on 10.13 |
| macOS 10.15 (Catalina) | Photoshop 2021 | 64-bit only |
| macOS 12 (Monterey) | Photoshop 2022 | Native Apple Silicon available |
| macOS 14 (Sonoma) | Photoshop 2024 | Requires Metal-capable GPU |
