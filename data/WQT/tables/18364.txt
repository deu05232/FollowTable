# Time-release formulations overview (focus on dosage form, not abbreviation lookup)

| Formulation type | Description | Example abbreviations |
| --- | --- | --- |
| Extended release | Releases drug over a longer period | ER, XR |
| Sustained release | Maintains drug levels for longer | SR |
| Controlled release | Designed for controlled drug delivery | CR |
| Delayed release | Releases drug later in GI tract | DR |
