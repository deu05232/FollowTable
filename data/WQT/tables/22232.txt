# Matrix and Vector Differentiation Identities (Derivative Rules)

| Expression | Differential Form | Derivative / Gradient Form | Conditions |
| --- | --- | --- | --- |
| y = a^T x | dy = a^T dx | ‚ąāy/‚ąāx = a | x is a vector |
| y = x^T a | dy = dx^T a | ‚ąāy/‚ąāx = a | Same result |
| y = x^T x | dy = 2 x^T dx | ‚ąāy/‚ąāx = 2x | Quadratic form |
| Y = A X | dY = A dX | ‚ąāY/‚ąāX = A | Linear map |
| y = tr(A^T X) | dy = tr(A^T dX) | ‚ąāy/‚ąāX = A | Trace trick |
| y = det(X) | dy = det(X) tr(X^{-1} dX) | ‚ąāy/‚ąāX = det(X) (X^{-1})^T | X invertible |
| y = x^T A x | dy = x^T(A + A^T) dx | ‚ąāy/‚ąāx = (A + A^T)x | A constant |
