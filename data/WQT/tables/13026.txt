# Parenteral Product and Container Terminology Abbreviations (Label-Focused)

| Abbreviation | Expansion (Packaging/Form Term) | Example Context |
|---|---|---|
| IVPB | Intravenous piggyback (secondary infusion bag) | Pharmacy compounding/administration setup |
| PFS | Prefilled syringe | Manufacturer packaging |
| SDV | Single-dose vial | Container type on labeling |
| MDV | Multi-dose vial | Container type on labeling |
| SVP | Small volume parenteral | Bag size classification |
| LVP | Large volume parenteral | Bag size classification |
| VIA | Vial adapter | Device component |
| NS | Normal saline (as diluent) | Often appears as diluent on labels |
| D5W | 5% dextrose in water (as diluent) | Bag labeling |
| gtt | Drops (drip factor reference) | IV set packaging specs |
| Y-site | Y-site connector | Tubing/connector term |
| spike | Spike port | Administration set component |
