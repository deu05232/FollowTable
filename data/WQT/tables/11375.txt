|---|---|
| Hx | history |
| Dx | diagnosis |
| Sx | symptoms |
| Tx | treatment |
| Rx | prescription/therapy |
| c/o | complains of |
| r/o | rule out |
| s/p | status post |
| WNL | within normal limits |
| f/u | follow up |
