# How to Join Popular Minecraft Servers (Step-by-Step Table)

| Step | What to do | Example server address |
| --- | --- | --- |
| 1 | Launch Minecraft Java Edition and ensure you are on the supported version (often 1.8–1.20). | — |
| 2 | Click **Multiplayer** and allow the firewall prompt if it appears. | — |
| 3 | Click **Add Server** and type a name you will recognize later. | Hypixel |
| 4 | Paste the server address into the address box, then click **Done**. | `mc.hypixel.net` |
| 5 | Alternatively use **Direct Connect** for a one-time join. | `play.cubecraft.net` |
