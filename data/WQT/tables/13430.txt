# Citra — Nintendo 3DS Game Compatibility Notes

| Game | Compatibility Result |
| --- | --- |
| Pokémon X | Playable; occasional frame drops |
| Animal Crossing: New Leaf | In-game; audio issues |
| The Legend of Zelda: Ocarina of Time 3D | Works; minor UI glitch |
| Fire Emblem: Awakening | Playable; rare crash reported |
| Mario Kart 7 | In-game; graphical issues on some GPUs |
| Super Mario 3D Land | Works well |
| Bravely Default | In-game; slowdowns in battles |
