# iPod Nano supported software and compatibility list

| iPod Nano generation | Supported iTunes version range | Supported macOS/Windows notes | File formats |
| --- | --- | --- | --- |
| 7th Gen | iTunes 11+ | USB sync required | AAC, MP3, Audible |
| 6th Gen | iTunes 10+ | requires USB 2.0 | AAC, MP3 |
| 5th Gen | iTunes 9+ | legacy drivers on older Windows | AAC, MP3 |
| 4th Gen | iTunes 8+ | older Macs may need updates | AAC, MP3 |
