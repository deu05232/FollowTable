# Texas Offense-Specific Felony Penalties (Selected Crimes)

| Offense | Statute reference | Felony classification | Maximum imprisonment | Maximum fine |
|---|---|---|---:|---:|
| Murder | Penal Code §19.02 | First-degree felony | Life | $10,000 |
| Aggravated robbery | Penal Code §29.03 | First-degree felony | Life | $10,000 |
| Burglary of habitation | Penal Code §30.02 | Second-degree felony | 20 years | $10,000 |
| Theft (high value) | Penal Code §31.03 | State jail felony | 2 years | $10,000 |
| Possession of controlled substance (small amount) | Health & Safety Code §481 | State jail felony | 2 years | $10,000 |
| Fraud/forgery (check) | Penal Code §32.21 | State jail felony | 2 years | $10,000 |
| Intoxication assault | Penal Code §49.07 | Third-degree felony | 10 years | $10,000 |
| Evading arrest with vehicle | Penal Code §38.04 | Third-degree felony | 10 years | $10,000 |
