# Typical ABV Ranges by Beverage Category

| Category | Example Styles | Typical ABV Range |
|---|---|---|
| Light beer | Light lager | 3.5–4.3% |
| Standard beer | Pale lager, pilsner | 4.5–5.5% |
| IPA | American IPA | 5.5–7.5% |
| Double IPA | Imperial IPA | 7.5–10.5% |
| Stout | Dry stout, oatmeal stout | 4.0–8.0% |
| Barleywine | English/American barleywine | 8.0–12.5% |
| Hard seltzer | Flavored seltzer | 4.0–6.0% |
| Wine | Table wine | 11.0–14.5% |
