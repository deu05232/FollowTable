# Timeline of Notable Browser Releases and Milestones

| Year | Browser / event | Major change | Platform focus | Notes |
| --- | --- | --- | --- | --- |
| 1994 | Netscape Navigator 1.0 | Commercial web browser debut | Windows/Mac | Early dominance |
| 1995 | Internet Explorer 1.0 | Bundled with Windows add-ons | Windows | Start of browser wars |
| 2003 | Safari 1.0 | Apple default browser | macOS | WebKit origins |
| 2004 | Firefox 1.0 | Extensions + tabbed browsing | Cross-platform | Mozilla revival |
| 2008 | Chrome 1.0 | Multi-process architecture | Cross-platform | Chromium ecosystem begins |
| 2015 | Microsoft Edge (EdgeHTML) | IE replacement | Windows | New default on Windows 10 |
| 2020 | Edge switches to Chromium | Engine change to Blink | Cross-platform | Compatibility shift |
| 2023 | Safari adds more web platform features | Incremental standards support | Apple platforms | Ongoing updates |
