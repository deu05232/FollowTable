# U.S. Army Enlisted Ranks (Titles and Abbreviations)

| Rank title | Common abbreviation | Typical role |
| --- | --- | --- |
| Private | PV1 | Entry-level soldier |
| Private First Class | PFC | Junior enlisted |
| Specialist | SPC | Technical/administrative track |
| Corporal | CPL | Junior NCO |
| Sergeant | SGT | Team leader |
| Staff Sergeant | SSG | Squad leader |
| Sergeant First Class | SFC | Platoon sergeant (assistant) |
| Master Sergeant | MSG | Senior NCO |
| First Sergeant | 1SG | Company senior enlisted |
| Sergeant Major | SGM | Battalion/brigade senior enlisted |
| Command Sergeant Major | CSM | Command senior enlisted |
| Sergeant Major of the Army | SMA | Senior enlisted advisor to CSA |
