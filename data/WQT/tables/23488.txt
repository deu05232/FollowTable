# New York Felony Sentencing Classes (General Framework)

| Felony class | Standard maximum prison term | Notes |
| --- | ---: | --- |
| Class A-I felony | Life imprisonment | Highest non-capital felony class |
| Class A-II felony | Life imprisonment | Typically covers the most serious controlled-substance felonies |
| Class B felony | 25 years | General maximum for class B felonies |
| Class C felony | 15 years | General maximum for class C felonies |
| Class D felony | 7 years | General maximum for class D felonies |
| Class E felony | 4 years | Lowest felony class |
| Unclassified felony | As specified by statute | Some felonies are not assigned a letter class |
| Attempted felony (generic rule) | Reduced maximum vs. completed offense | Maximum is generally lower than the completed felony (varies by class) |
