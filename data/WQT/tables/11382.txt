# Medication formulation and release abbreviations used on pharmacy labels

| Abbreviation | Meaning |
| --- | --- |
| IR | immediate release |
| SR | sustained release |
| ER | extended release |
| DR | delayed release |
| CR | controlled release |
| LA | long-acting |
| SA | short-acting |
| EC | enteric-coated |
| ODT | orally disintegrating tablet |
| PR | prolonged release |
