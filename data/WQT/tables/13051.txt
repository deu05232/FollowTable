# Common Windows Alt codes for currency and punctuation symbols

| Alt code | Symbol | Name/description |
|---|---|---|
| Alt 0128 | € | Euro sign |
| Alt 0163 | £ | Pound sign |
| Alt 0165 | ¥ | Yen sign |
| Alt 0162 | ¢ | Cent sign |
| Alt 0177 | ± | Plus-minus |
| Alt 0176 | ° | Degree sign |
| Alt 0174 | ® | Registered sign |
