# Standard Adult BP Categories by Systolic Diastolic Cutoffs (mm Hg) — Interpretation Guide

| Classification | Systolic range (mm Hg) | Diastolic range (mm Hg) | Rule for single measurement |
|---|---:|---:|---|
| Optimal | < 120 | and < 80 | Classify as optimal only if both meet cutoffs |
| Normal | 120–129 | and/or 80–84 | If either number falls here and none are higher |
| High-normal | 130–139 | and/or 85–89 | If either number falls here and none are higher |
| Hypertension (Grade 1) | 140–159 | and/or 90–99 | Grade is determined by the higher of systolic/diastolic category |
| Hypertension (Grade 2) | 160–179 | and/or 100–109 | Use this if either number is in this band |
| Hypertension (Grade 3) | ≥ 180 | and/or ≥ 110 | Highest routine category by threshold |
| Isolated systolic hypertension | ≥ 140 | and < 90 | Applies when systolic is high but diastolic is not |
| Isolated diastolic hypertension | < 140 | and ≥ 90 | Applies when diastolic is high but systolic is not |
