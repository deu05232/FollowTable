  <tr><td>Test Sub-item</td><td>Terminal Vendor Self-service Interface Terminal Management</td></tr>
  <tr><td>Test Objective</td><td>Verify the terminal management functions of the terminal vendor through the portal</td></tr>
  <tr><td>Preconditions</td><td>1. Log in to the system using a terminal vendor administrator account; 2. The page displays normally; 3. Input fields and buttons can be operated normally; 4. The system is normally connected to the server; 5. At least two terminals have been successfully connected under this terminal vendor administrator account 1. Log in to the system, enter the terminal vendor self-service interface, select Terminal Management, and the interface automatically displays the current industry terminal vendor management</td></tr>
  <tr><td>Test Steps</td><td>There is already a terminal list under the account; 2. Select Terminal Information Query to enter the query interface; 3. Enter query item information such as terminal serial number, terminal type, and terminal static status on the interface, then click OK; 4. Return to the terminal vendor self-service - terminal management interface, select Terminal Working Status Query to enter the query interface; 5. Enter query item information such as terminal serial number on the interface, then click OK; 6. Return to the terminal vendor self-service - terminal management interface, select a terminal record, then select Terminal Working Status Query; 7. Return to the terminal vendor self-service — terminal management interface, select a terminal record, then select Terminal Parameter Query to enter the terminal parameter display interface; 8. Click Settings, modify each parameter on the interface, then click Save; 9. Return to the terminal vendor self-service - terminal management interface, select the previously modified terminal record, then select Terminal Parameter Query to enter the terminal parameter display interface; 10. Return to the terminal vendor self-service - terminal management interface, select Terminal Fault Query to enter the query interface; 11. Enter query item information such as terminal serial number on the interface, then click OK; 12. Return to the terminal vendor self-service - terminal management interface, select a terminal record, and choose to view the terminal status; 13. Return to the terminal vendor self-service — terminal management interface, select one or more terminal records, then select Terminal Control to enter the terminal remote control interface; 14. After modifying the terminal status, return to the terminal vendor self-service - terminal management interface to view the status of one terminal; 15. Return to the terminal vendor self-service - terminal management interface, select one or more terminal records, then select Status Detection to enter the status detection settings interface; 16. Create a terminal status detection task; 17. After a period of time (greater than the reporting period configured for the selected task), view the terminal reporting information record for one of the terminals specified by the task</td></tr>
  <tr><td>Pass Criteria</td><td>Support managing terminals through the portal, including terminal information query, terminal working status query, terminal parameter query and remote configuration, terminal fault query, terminal remote control, terminal status monitoring, etc.</td></tr>
