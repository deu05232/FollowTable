    <tr><td>PP (Polypropylene)</td><td>Bumpers, interior trims, wheel-arch liners</td><td>rPP pellets / injection-molded parts</td><td>Manual sorting → shredding → washing → drying → mixing → extrusion → pelletizing</td><td>Paint removal and melt filtration to control black specks</td></tr>
    <tr><td>ABS</td><td>Instrument panel components, pillar covers, console parts</td><td>rABS pellets / molded housings</td><td>Identification sorting → shredding → washing → drying → mixing → extrusion → pelletizing</td><td>Strict sorting to avoid PP cross-contamination affecting impact strength</td></tr>
    <tr><td>PC/ABS (blend)</td><td>Decorative interior parts, ventilation components</td><td>Recycled blend pellets</td><td>Part-level sorting → shredding → washing → drying → mixing → extrusion → pelletizing</td><td>Keep blend stream separate from neat ABS to stabilize melt flow</td></tr>
    <tr><td>HDPE/PE</td><td>Washer fluid reservoirs, ducts, covers</td><td>rPE pellets / extruded profiles</td><td>Sorting → shredding → washing → drying → mixing → extrusion → pelletizing</td><td>Density-based separation helps remove PP and elastomer attachments</td></tr>
    <tr><td>PVC</td><td>Wire/cable sheathing, interior skins (where present)</td><td>rPVC pellets / molded non-appearance items</td><td>Dedicated sorting → shredding → washing → drying → mixing → extrusion → pelletizing</td><td>Dedicated line recommended to prevent chlorine carryover into other streams</td></tr>
    <tr><td>HIPS/PS</td><td>Interior liners, small housings</td><td>rHIPS pellets</td><td>Sorting → shredding → washing → drying → mixing → extrusion → pelletizing</td><td>Avoid PC/ABS mixing; melt filtration improves surface quality</td></tr>
    <tr><td>Mixed Thermoplastics (for downcycled use)</td><td>Small mixed interior plastics not economically separable</td><td>Mixed recycled pellets / compression-molded boards</td><td>Coarse sorting → shredding → washing → drying → mixing → extrusion → pelletizing (or direct compression molding)</td><td>Used for non-appearance products where polymer purity is less critical</td></tr>
