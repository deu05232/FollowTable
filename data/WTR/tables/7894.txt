# iPod Manager Software Compatibility Matrix (Desktop Apps)

| Software | Platform | License | iPod Classic Support | iPod Nano Support | iPod Shuffle Support | iPod Touch Support | Notes |
|---|---|---|---|---|---|---|---|
| iTunes | Windows/macOS | Proprietary | Yes | Yes | Yes | Yes | Official Apple sync tool |
| Rhythmbox | Linux | GPL | Yes (limited) | Yes | Yes | Partial | Requires device rules |
| gtkpod | Linux/macOS | GPL | Yes | Yes | Yes | No | Best for older iPods |
| Amarok | Linux | GPL | Partial | Partial | Partial | No | Depends on libgpod |
| Winamp | Windows | Proprietary | Yes | Yes | Yes | No | Older plugin-based support |
| MediaMonkey | Windows | Proprietary | Yes | Yes | Yes | No | Good tag management |
| CopyTrans Manager | Windows | Proprietary | Yes | Yes | Yes | Partial | Focused on music transfer |
| Banshee | Linux | MIT | Partial | Partial | Partial | No | Development discontinued |
