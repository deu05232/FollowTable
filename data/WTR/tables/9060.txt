# Google Shopping   Checkout APIs — Directory of eCommerce Endpoints (Excerpt)

| API / Service | Provider | Primary use | Auth | Typical response | Region | Notes |
|---|---|---|---|---|---|---|
| Google Shopping Content API | Google | Manage product feeds | OAuth 2.0 | JSON | Global | Merchant Center integration |
| Google Shopping Search API | Google | Query product listings | API key | JSON | Limited | Deprecated in some contexts |
| Google Checkout | Google | Payments (legacy) | Token | XML/JSON | Legacy | Historical service |
| Google Merchant Promotions | Google | Promo submission | OAuth 2.0 | JSON | Select | Requires approval |
| Best Buy Reviews API | Best Buy | Retrieve review text | API key | JSON | US | Not a ratings table |
| Bazaarvoice | Third-party | Hosted reviews | OAuth/API key | JSON | Global | Collects reviews for retailers |
| Commission Junction | CJ | Affiliate links | API key | JSON | Global | Not product evaluation |
| Shopzilla | Shopzilla | Product search | API key | JSON | US | Aggregates listings |
