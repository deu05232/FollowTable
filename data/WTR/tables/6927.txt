# Moon Phases and Related Lunar Month Metrics (Synodic vs Sidereal vs Anomalistic vs Draconic)

| Lunar cycle / month type | Definition (what repeats) | Typical length (days) | Primary use/context |
|---|---|---:|---|
| **Synodic month** | Phase-to-phase repeat (New Moon to New Moon) | 29.53059 | Moon phases, lunations |
| **Sidereal month** | Return to same star background position | 27.32166 | Orbital motion relative to stars |
| **Anomalistic month** | Perigee-to-perigee (distance cycle) | 27.55455 | Apparent size changes (supermoon/perigee) |
| **Draconic month** | Node-to-node (orbit-plane crossing) | 27.21222 | Eclipse seasons (node alignment) |
| **Tropical month** | Return to same ecliptic longitude | 27.32158 | Reference-frame/precession context |
| **Lunation numbering** | Counting synodic cycles from an epoch | ~29.53 each | Calendars/astronomy indexing |
| **Phase quarter spacing** | New→First Quarter→Full→Last Quarter | ~7.38 each (avg) | Phase timing summaries |
