# BitTorrent clients and their key attributes

| Client | Platforms | License | Remote control / Web UI | Notable features |
|---|---|---|---|---|
| qBittorrent | Windows, macOS, Linux | GPL | Yes | Built-in search, no-ads UI |
| Transmission | macOS, Linux, *BSD | GPL | Yes | Lightweight, simple interface |
| Deluge | Windows, macOS, Linux | GPL | Yes | Plugin-based, daemon mode |
| rTorrent | Linux, *BSD | GPL | Yes (via ruTorrent/SCGI) | Console-based, very low overhead |
| BitComet | Windows | Proprietary | Yes | Long-term seeding features |
| Vuze | Windows, macOS, Linux | GPL (with optional components) | Yes | Rich UI, plugins, media features |
