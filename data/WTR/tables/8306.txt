# Ibanez 8-String Models — Feature-to-Benefit Map (Construction & Hardware)

| Feature Area | Concrete Spec (Ibanez 8-string examples) | What it changes in use (benefit tied to the spec) | Comparison/notes |
|---|---|---|---|
| Scale length | 27.0 in baritone scale | Higher string tension on low strings; clearer low notes | Useful for drop tunings on 8-string |
| Neck construction | Bolt-on neck joint | Snappier attack; easier neck service/adjustments | Common across many modern Ibanez metal-oriented models |
| Neck woods | 5-piece maple/walnut neck | Improved stiffness; may reduce neck movement with humidity | Multi-piece laminations are a stability-focused build choice |
| Fretboard | Rosewood or jatoba fretboard | Smoother feel; slightly warmer perceived attack vs bright maple | Material choice affects feel more than “tone” for many players |
| Fingerboard radius | 400–430 mm radius (flat-ish) | Easier wide bends; supports lower action without fretting out | A measurable setup spec that differs from vintage guitars |
| Frets | 24 frets, jumbo size | More upper range; jumbo height can feel faster for legato | Setup may require lighter touch to avoid sharping notes |
| Pickups | Passive humbuckers or active (high-output) set | Higher output, tighter low end; lower noise vs single coils | Active sets require a battery cavity |
| Pickup layout | H-H or H-S-H wiring options | More voicing flexibility (split/parallel positions if wired) | Switching varies by model (3-way vs 5-way) |
| Bridge | Fixed bridge or double-locking tremolo | Fixed: simpler tuning stability; Trem: pitch effects + locking stability | Trem systems add setup complexity (springs, leveling) |
| Nut type | Locking nut (trem models) vs standard nut (fixed models) | Locking nut increases tuning stability under trem use | String changes slower on locking systems |
| String spacing | Wider 8-string bridge spacing (typical 8-string spec) | More room for right-hand picking accuracy | Exact mm varies by bridge family |
| Factory setup baseline | .009–.065 or similar 8-string set (varies) | Establishes feel/tension baseline; many go heavier for lower tunings | String gauge is a practical spec for comparison |
| Body wood | Ash, basswood, or mahogany-family woods | Weight/attack differences; ash often feels snappier | Spec helps anticipate weight and response |
