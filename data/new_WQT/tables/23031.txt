# Companion plants that repel common vegetable-garden pests

| Plant | Pests repelled |
| --- | --- |
| Basil | mosquitoes, houseflies |
| Nasturtium | aphids, whiteflies |
| Lavender | moths, fleas |
| Rosemary | cabbage moth, carrot fly |
| Sage | cabbage moth |
| Catnip | mosquitoes, ants |
| Garlic | aphids, spider mites |
