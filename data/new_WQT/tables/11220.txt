# UN ECE and National Child Restraint Product Standards (Overview)

| Standard/Regulation | Region | Scope | Key Testing Focus | Typical Label/Marking |
|---|---|---|---|---|
| FMVSS 213 | United States | Child restraint systems | Crash performance, buckles, flammability | "This restraint is certified for use in motor vehicles" |
| UN R44/04 | UN/ECE (Europe and others) | Child seats by weight group | Dynamic crash tests, installation checks | "E" in a circle + country code |
| UN R129 (i-Size) | UN/ECE (Europe and others) | Child seats by height | Side impact, ISOFIX fit, head excursion | "i-Size" approval label |
| AS/NZS 1754 | Australia/New Zealand | Child restraints | Design rules, performance tests | Tick mark label |
| CMVSS 213 | Canada | Child restraints | Performance + labeling | Maple-leaf compliance mark |
| GB 27887 | China | Child restraints | Crash + material requirements | CCC mark |
| JIS D 0401 | Japan | Child restraints | Crash performance | JIS compliance label |
