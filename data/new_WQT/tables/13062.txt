# Windows Alt Codes (Alt+Number) for Punctuation, Quotation, and Dashes

| Alt code (hold Alt + type) | Symbol produced | Description |
|---:|:---:|---|
| Alt+0134 | † | Dagger |
| Alt+0135 | ‡ | Double dagger |
| Alt+0145 | ‘ | Left single quotation mark |
| Alt+0146 | ’ | Right single quotation mark |
| Alt+0147 | “ | Left double quotation mark |
| Alt+0148 | ” | Right double quotation mark |
| Alt+0150 | – | En dash |
| Alt+0151 | — | Em dash |
| Alt+0139 | ‹ | Single left-pointing angle quote |
| Alt+0155 | › | Single right-pointing angle quote |
| Alt+0171 | « | Left-pointing double angle quote |
| Alt+0187 | » | Right-pointing double angle quote |
