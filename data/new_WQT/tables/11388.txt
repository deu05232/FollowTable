# Route and administration abbreviations commonly used in medication orders

| Abbreviation | Meaning |
| --- | --- |
| PO | by mouth (oral) |
| SL | sublingual |
| PR | rectal |
| IM | intramuscular |
| IV | intravenous |
| SQ | subcutaneous |
| INH | inhalation |
| TOP | topical |
| OPH | ophthalmic |
| AU | both ears |
