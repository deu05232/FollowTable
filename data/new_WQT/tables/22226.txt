# Derivative rules with mixed content  combination rules plus common function derivatives

| Item type | Expression | Derivative |
|---|---|---|
| Sum rule | \(f+g\) | \(f'+g'\) |
| Product rule | \(fg\) | \(fg'+f'g\) |
| Exponential derivative | \(e^x\) | \(e^x\) |
| Trig derivative | \(\sin x\) | \(\cos x\) |
| Log derivative | \(\ln x\) | \(1/x\) |
| Power rule | \(x^n\) | \(n x^{n-1}\) |
