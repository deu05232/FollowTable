# Common prescription (sig) abbreviations for medication timing and frequency

| Abbreviation | Meaning |
| --- | --- |
| qd | once daily |
| bid | twice daily |
| tid | three times daily |
| qid | four times daily |
| qhs | every night at bedtime |
| prn | as needed |
| stat | immediately |
| ac | before meals |
| pc | after meals |
| q4h | every 4 hours |
