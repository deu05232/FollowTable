# SQL Server 2008 hardware and software requirements

| Hardware | Minimum Hardware Requirements 32-Bit |
| --- | --- |
| CPU | Pentium III-compatible processor or faster. 1GHz minimum. Recommended 2GHz or faster. |
| Memory (RAM) | 512MB minimum, 2GB or more recommended. Report Server will use a maximum of 3GB (with /3GB switch in boot.ini). |
