# Emulators vs. Games  Quick Picks for Popular Titles

| Game | Original System | Recommended Emulator | Host OS | Works? | Common Fix |
|---|---|---|---|---|---|
| Pokémon Emerald | Game Boy Advance | VisualBoyAdvance | Windows | Yes | Set save type to Flash 128K |
| The Legend of Zelda: Ocarina of Time | Nintendo 64 | Project64 | Windows | Mostly | Use GlideN64 plugin |
| Gran Turismo 2 | PlayStation 1 | DuckStation | Windows / Linux | Yes | Enable PGXP for stability |
| Persona 3 Portable | PSP | PPSSPP | Android / Windows | Yes | Switch to Vulkan renderer |
| Super Metroid | SNES | bsnes | Windows / macOS | Yes | Use cycle-accurate core |
| Metal Slug X | Neo Geo | FinalBurn Neo | Windows | Yes | Set audio to 48kHz |
| Animal Crossing: Wild World | Nintendo DS | DeSmuME | Windows / macOS | Partial | Enable microphone emulation |
