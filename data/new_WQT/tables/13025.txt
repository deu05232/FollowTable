# Oral Solid Dose Product Markings and Manufacturing Abbreviations

| Abbreviation | Expansion (Manufacturing/Packaging Meaning) | Notes |
|---|---|---|
| GMP | Good Manufacturing Practice | Regulatory/manufacturing context |
| NDC | National Drug Code | Product identifier on packaging |
| LOT | Lot number | Traceability on packaging |
| EXP | Expiration date | Packaging date field |
| blister | Blister pack | Packaging type |
| HDPE | High-density polyethylene (bottle material) | Container material |
| CRC | Child-resistant closure | Closure type |
| TE | Tamper-evident | Packaging feature |
| Rx only | Prescription only | Labeling classification |
| OTC | Over-the-counter | Retail classification |
| imprint | Tablet imprint | Product identification |
| comp | Comparator product | Bioequivalence/packaging docs |
