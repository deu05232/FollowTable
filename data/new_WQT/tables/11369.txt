# Abbreviations Used in Orders  Routes, Procedures, and Dosing

| Abbreviation | Meaning |
|---|---|
| PO | by mouth |
| IV | intravenous |
| SQ | subcutaneous |
| stat | immediately |
| q8h | every 8 hours |
| PRN | as needed |
| NGT | nasogastric tube |
| CT | computed tomography |
| ECG | electrocardiogram |
| ORIF | open reduction and internal fixation |
| CABG | coronary artery bypass graft |
| LP | lumbar puncture |
