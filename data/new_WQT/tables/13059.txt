# Typing special characters using various keyboard shortcuts (Alt codes mixed with other methods)

| Character | Windows Alt code | Mac shortcut | Unicode code point | Notes |
|---|---|---|---|---|
| © | Alt 0169 | Option+G | U+00A9 | Copyright |
| ® | Alt 0174 | Option+R | U+00AE | Registered |
| ™ | Alt 0153 | Option+2 | U+2122 | Trademark |
| € | Alt 0128 | Option+Shift+2 | U+20AC | Euro |
| ° | Alt 0176 | Option+Shift+8 | U+00B0 | Degree |
| ± | Alt 0177 | Option+Shift+= | U+00B1 | Plus-minus |
| µ | Alt 0181 | Option+M | U+00B5 | Micro sign |
