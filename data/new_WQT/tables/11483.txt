# UK National Lottery Lotto — How to Play & Winning Rules

| Topic | Description |
| --- | --- |
| Game format | Choose 6 numbers from 1–59 |
| Draw frequency | Twice weekly |
| Jackpot rule | Match all 6 main numbers |
| Bonus ball | Used for an additional prize category |
| Prize tiers | Multiple match categories (e.g., 2, 3, 4, 5, 5+Bonus, 6) |
| Rollovers | Jackpot rolls when no one matches all 6 |
| Free play | Some lower-tier wins may be paid as a free Lucky Dip (operator-dependent) |
