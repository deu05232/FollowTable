# Vendor-Default Administrative Distance (AD) Values by Route Source (Cisco IOS Reference)

| Route Source / Protocol | Default AD | Notes / Variants |
|---|---:|---|
| Connected (directly attached) | 0 | Directly connected interface routes |
| Static | 1 | Standard static route |
| Static (floating) | 2–254 | Any static AD set higher than primary route |
| EIGRP summary | 5 | Summary route generated by EIGRP |
| eBGP | 20 | External BGP |
| EIGRP internal | 90 | Intra-AS EIGRP routes |
| IGRP | 100 | Legacy protocol |
| OSPF | 110 | Includes OSPF intra/inter-area by default |
| IS-IS | 115 | Integrated IS-IS |
| RIP | 120 | RIPv1/RIPv2 |
| EIGRP external | 170 | Routes redistributed into EIGRP |
| iBGP | 200 | Internal BGP |
| Unknown / unusable | 255 | Route never installed due to distance |

> Optional detail (common sub-types):

| OSPF Route Type | Default AD | Notes |
|---|---:|---|
| OSPF Intra-area (O) | 110 | Same default as OSPF overall |
| OSPF Inter-area (O IA) | 110 | Same default as OSPF overall |
| OSPF External (E1/E2) | 110 | Same default as OSPF overall |
| OSPF NSSA (N1/N2) | 110 | Same default as OSPF overall |
