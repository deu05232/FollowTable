# Adult BMI Classification Table (WHO CDC-aligned) — BMI Ranges and Weight-Status Labels

| BMI (kg/m²) | Weight-status label | Common description |
|---|---|---|
| < 16.0 | Underweight | Severe thinness |
| 16.0–16.9 | Underweight | Moderate thinness |
| 17.0–18.4 | Underweight | Mild thinness |
| 18.5–24.9 | Normal weight | Healthy range |
| 25.0–29.9 | Overweight | Pre-obesity |
| 30.0–34.9 | Obesity (Class I) | Moderate obesity |
| 35.0–39.9 | Obesity (Class II) | Severe obesity |
| ≥ 40.0 | Obesity (Class III) | Very severe (morbid) obesity |
