# Symmetry Operations by Group  Rotations, Reflections, Inversions (Not Shape-Centric)

| Symmetry group label | Primary generators | Rotation elements (count/type) | Reflection / inversion elements | Typical application |
| --- | --- | --- | --- | --- |
| C1 | identity | 1 (E) | none | Asymmetric objects |
| C2 | 180° rotation | 2 (E, C2) | none | Molecules with a single 2-fold axis |
| C3 | 120° rotations | 3 (E, C3, C3^2) | none | Trigonal arrangements |
| D2 | three C2 axes | 4 total proper rotations | none | Rectangular solids / frameworks |
| D3 | C3 + three C2 | 6 total proper rotations | none | Triangular prisms (rotation-only view) |
| C2v | C2 + mirrors | 2 proper rotations | 2 mirrors | Planar molecules with one C2 axis |
| D4h | C4 + C2 + mirrors + inversion | 8 proper rotations | mirrors + inversion present | Square-planar coordination |
| Ih | icosahedral + inversion | 60 proper rotations | + inversion doubles full order | Fullerenes / viral capsids |
