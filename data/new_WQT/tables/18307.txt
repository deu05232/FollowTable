# Comparison of web browsers — Feature and standards compatibility

| Browser | Rendering engine | JavaScript engine | HTML5 baseline support | CSS3 baseline support | Extensions/Add-ons | Notes |
| --- | --- | --- | --- | --- | --- | --- |
| Google Chrome | Blink | V8 | High | High | Yes (Chrome Web Store) | Frequent stable releases; strong DevTools |
| Mozilla Firefox | Gecko | SpiderMonkey | High | High | Yes (AMO) | Strong privacy controls; multi-platform |
| Microsoft Edge | Blink | V8 | High | High | Yes (Edge Add-ons) | Chromium-based; tight Windows integration |
| Apple Safari | WebKit | JavaScriptCore | High | High | Limited (Safari Extensions) | Best integration on Apple platforms |
| Opera | Blink | V8 | High | High | Yes (incl. Chrome extensions) | Built-in VPN/ad blocker in some builds |
| Brave | Blink | V8 | High | High | Yes (Chrome Web Store) | Built-in tracker blocking by default |
| Vivaldi | Blink | V8 | High | High | Yes (Chrome Web Store) | Heavy UI customization; power-user features |
