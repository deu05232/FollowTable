# Outlook.com account setup checklist (includes ports, but mixed with general fields)

| Setup item | Value |
| --- | --- |
| Display name | Your name |
| Email address | yourname@outlook.com |
| Username | full email address |
| Password | your Microsoft account password |
| Incoming server | imap-mail.outlook.com |
| IMAP port / security | 993 / SSL-TLS |
| Outgoing server | smtp-mail.outlook.com |
| SMTP port / security | 587 / TLS (STARTTLS) |
