# Common Garden Plants and the Specific Pests They Repel

| Plant | Pests repelled (specific examples) |
| --- | --- |
| Lavender (Lavandula spp.) | mosquitoes, moths, fleas |
| Rosemary (Salvia rosmarinus) | cabbage moths, carrot fly, mosquitoes |
| Basil (Ocimum basilicum) | mosquitoes, houseflies, thrips |
| Peppermint (Mentha × piperita) | ants, aphids, cabbage loopers |
| Sage (Salvia officinalis) | cabbage moths, flea beetles, carrot rust fly |
| Nasturtium (Tropaeolum majus) | aphids, whiteflies, squash bugs |
| Garlic (Allium sativum) | aphids, spider mites, Japanese beetles |
| Catnip (Nepeta cataria) | mosquitoes, cockroaches, ants |
