# Modpack Launcher-Based Minecraft Servers (Connection via Setup Steps)

| Rank | Pack/Launcher | Required Steps (primary content) | Server Entry |
| --- | --- | --- | --- |
| 1 | Technic Launcher | Install Technic → Search pack name → Install → Launch → Multiplayer (server is pre-added). | “Included in pack” |
| 2 | CurseForge App | Import profile → Ensure correct Forge version → Allocate RAM → Launch → Accept resource packs. | “Auto-join button” |
| 3 | ATLauncher | Create instance → Download dependencies → Launch → Use in-pack server list menu item. | “Shown in menu” |
| 4 | GDLauncher | Add modpack by URL → Wait for download → Start → Use the packaged server list. | “Bundled” |
| 5 | MultiMC/Prism | Add instance from zip → Configure Java → Launch → Use the provided multiplayer server preset. | “Preset entry” |
| 6 | Fabric Installer | Install Fabric → Drop mods → Launch → Make sure versions match → Use the server address from README. | “In README” |
| 7 | Custom Client | Download our client → Log in → Click Play → It connects automatically; manual IP is not supported. | “No manual address” |
