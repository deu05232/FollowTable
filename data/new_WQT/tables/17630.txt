# Outlook.com   Hotmail Service Timeline and Renaming History (Hotmail, Windows Live Mail, Outlook.com)

| Year | Brand/Service name | Later name | URL/Platform | What changed | Notes |
| --- | --- | --- | --- | --- | --- |
| 1996 | Hotmail | MSN Hotmail | hotmail.com | Launch of free webmail | Early webmail adoption |
| 2005 | MSN Hotmail | Windows Live Hotmail | live.com | Integration into Windows Live | Branding shift |
| 2007 | Windows Live Mail (client) | Windows Live Essentials Mail | Desktop app | Client software distribution | Not server settings |
| 2011 | Hotmail (web) | Outlook.com preview | outlook.com | New UI and features | Preview phase |
| 2013 | Outlook.com | Outlook.com | outlook.com | Migration completed | Hotmail aliases retained |
| 2014 | Microsoft account | Microsoft account | account.microsoft.com | Identity system consolidation | Login unification |
| 2020 | Outlook.com | Outlook on the web | outlook.live.com | Product naming updates | Still webmail |
| 2023 | Outlook (consumer) | New Outlook for Windows | Windows app | New client experience | Separate from protocol details |
