# U.S. Passport Application Fees (Routine Base Fee vs Expedited Surcharge)

| Passport request type | Base fee (routine processing) | Expedited processing add-on | Notes |
| --- | ---: | ---: | --- |
| First-time adult passport book (age 16+) | $130 | $60 | Add the expedited surcharge to the base fee for faster processing |
| First-time minor passport book (under 16) | $100 | $60 | Expedited fee is the same add-on across request types |
| Adult renewal (passport book) | $130 | $60 | Renewal uses the same expedited add-on concept |
| Replacement: lost passport book (adult) | $130 | $60 | Replacement priced like a new book plus the expedite add-on |
| Replacement: stolen passport book (minor) | $100 | $60 | Expedited is listed as a surcharge, not a separate product |
| Passport card only (adult) | $30 | $60 | Expedited add-on can be applied to the card request |
| Passport card only (minor) | $15 | $60 | Base vs expedite shown side-by-side for easy comparison |
| Name change / data correction with re-issue | $0 | $60 | Base is $0 when eligible; expedite is still an optional add-on |
