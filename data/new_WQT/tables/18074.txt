# Spine MRI CPT Codes by Region and Contrast (Cervical, Thoracic, Lumbar, Sacrum Coccyx)

| Spine Region | CPT Code (No Contrast) | CPT Code (With Contrast) | CPT Code (With & Without Contrast) | Notes |
| --- | --- | --- | --- | --- |
| Cervical spine | 72141 | 72142 | 72156 | Standard diagnostic MRI of the neck/cervical spine |
| Thoracic spine | 72146 | 72147 | 72157 | Mid-back/thoracic spine MRI options by contrast use |
| Lumbar spine | 72148 | 72149 | 72158 | Low-back/lumbar spine MRI options by contrast use |
| Sacrum/Coccyx | 72195 | 72196 | 72197 | Often ordered for sacral pain, mass, or SI-region evaluation |
| Entire spine (screening) | 72141 + 72146 + 72148 | 72142 + 72147 + 72149 | 72156 + 72157 + 72158 | Typically billed as separate regional spine MRIs rather than one single CPT |
| Cervical + Thoracic | 72141 + 72146 | 72142 + 72147 | 72156 + 72157 | Combined orders generally map to multiple region-specific CPT codes |
| Thoracic + Lumbar | 72146 + 72148 | 72147 + 72149 | 72157 + 72158 | Combined orders generally map to multiple region-specific CPT codes |
| Cervical + Lumbar | 72141 + 72148 | 72142 + 72149 | 72156 + 72158 | Combined orders generally map to multiple region-specific CPT codes |
