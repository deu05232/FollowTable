# Metallocene
> Classification

| Class | Example |
| --- | --- |
| Alcohol | Ethanol |
| Amino acids | Glutamic acid, aspartic acid |
| Nucleotides | 5' guanylic acid |
| Antioxidants | Isoascorbic acid |
| Organic acids | Acetic acid, lactic acid |
| Polyols | Glycerol |
| Vitamins | B 2 |
| Formula | Description |
| [( η 5 -C 5 H 5 ) 2 M] | Symmetrical, classical 'sandwich' structure |
| [( η 5 -C 5 H 5 ) 2 ML x ] | Bent or tilted Cp rings with additional ligands, L |
| [( η 5 -C 5 H 5 )ML x ] | Only one Cp ligand with additional ligands, L ('piano-stool' structure) |
