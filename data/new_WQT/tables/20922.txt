# # GCSE Subjects and Typical Assessment

| Subject | GCSE assessment style | Typical components |
| --- | --- | --- |
| Mathematics | Exams | Calculator & non-calculator papers |
| English Language | Exams + speaking endorsement | Reading, writing, spoken language |
| Biology | Exams | Papers plus required practicals |
| History | Exams | Source analysis and extended essays |
| Art and Design | Coursework portfolio + exam project | Portfolio, externally set assignment |
