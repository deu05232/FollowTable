# RFC 1918 private IPv4 blocks (CIDR form)

| Block (CIDR) | Common name | Notes |
| --- | --- | --- |
| 10.0.0.0/8 | 24-bit block | Private use inside organizations |
| 172.16.0.0/12 | 20-bit block | Private use (172.16.0.0–172.31.255.255) |
| 192.168.0.0/16 | 16-bit block | Private use in home/small networks |
