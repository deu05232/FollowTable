# Windows 10-Compatible Intel CPU Families (Desktop and Mobile Overview)

| CPU Family | Codename | Example SKUs | Launch window | Typical socket/package | Integrated graphics | Notes on Windows 10 support |
|---|---|---|---|---|---|---|
| Core i3 (2nd Gen) | Sandy Bridge | i3-2100, i3-2310M | 2011 | LGA1155 / rPGA988B | HD Graphics 2000/3000 | Often runs Windows 10 with legacy drivers depending on OEM |
| Core i5 (2nd Gen) | Sandy Bridge | i5-2500K, i5-2520M | 2011 | LGA1155 / rPGA988B | HD 2000/3000 | Broad Win10 compatibility; chipset varies by platform |
| Core i7 (3rd Gen) | Ivy Bridge | i7-3770, i7-3630QM | 2012 | LGA1155 / rPGA988B | HD 2500/4000 | Generally solid Windows 10 support |
| Core i5 (4th Gen) | Haswell | i5-4570, i5-4300U | 2013 | LGA1150 / BGA1168 | HD 4600 / Iris | Very common Windows 10 laptops/desktops |
| Core i7 (6th Gen) | Skylake | i7-6700, i7-6500U | 2015 | LGA1151 / BGA1356 | HD 530 | Strong Windows 10 driver availability |
| Atom / Celeron (Braswell) | Braswell | N3050, N3150 | 2015 | BGA | HD Graphics | Windows 10 used widely on low-power devices |
| Core i5/i7 (8th Gen) | Kaby Lake-R | i5-8250U, i7-8550U | 2017 | BGA | UHD 620 | Typically Windows 10 OEM default |
| Core (10th Gen) | Comet Lake | i5-10210U, i7-10700 | 2019-2020 | LGA1200 / BGA | UHD 630 | Windows 10 support is mainstream |
