# State Rules on Accepting a Federal Extension (Not Actual Filing Deadlines)

| State | Accepts Federal Extension? | Separate State Extension Required? | How to Claim Extension | Notes |
|---|---|---|---|---|
| Idaho | Yes | No | Automatic with federal extension | Payment may still be due by original date |
| Illinois | Yes | No | Automatic if no balance due | File state return by extended date |
| Indiana | Yes | Sometimes | Attach a copy of federal extension | Varies by residency status |
| Iowa | Yes | No | Automatic | Must pay expected tax by original due date |
| New Jersey | No | Yes | File NJ extension form | Federal extension alone is not sufficient |
| Pennsylvania | Yes | No | Automatic | Extension is for filing only |
| Virginia | Yes | No | Automatic | Penalties may apply if not paid timely |
| Missouri | Yes | No | Automatic | Keep federal extension for records |
