# TeX LaTeX Input Notation to Rendered Symbols

| Category | TeX Input | Rendered Symbol | Meaning |
|---|---|:---:|---|
| Relations | \le | ≤ | less-than or equal |
| Relations | \ge | ≥ | greater-than or equal |
| Logic | \neg | ¬ | negation |
| Arrows | \Rightarrow | ⇒ | implies |
| Sets | \in | ∈ | element of |
| Sets | \subseteq | ⊆ | subset or equal |
| Greek | \alpha | α | greek alpha |
| Greek | \pi | π | greek pi |
