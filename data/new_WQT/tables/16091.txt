# Rotational Symmetry of Common 2D Shapes (Given as Angles per Repeat)

| 2D Shape | Smallest rotation that maps the shape onto itself (degrees) | Notes |
| --- | ---: | --- |
| Scalene triangle | 360° | No non-trivial rotational symmetry |
| Isosceles triangle | 360° | Only returns after full turn |
| Equilateral triangle | 120° | Repeats every 120° |
| Rectangle (non-square) | 180° | Half-turn symmetry |
| Square | 90° | Quarter-turn symmetry |
| Regular pentagon | 72° | Repeats every 72° |
| Regular hexagon | 60° | Repeats every 60° |
| Parallelogram (generic) | 180° | Half-turn symmetry |
| Rhombus (non-square) | 180° | Half-turn symmetry |
| Circle | Any angle | Continuous rotational symmetry |
