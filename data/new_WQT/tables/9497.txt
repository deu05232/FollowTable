# Companion Plants  Plant-by-Plant Pest Repellency Reference

| Plant | Pests repelled (named pests) |
| --- | --- |
| Marigold (Tagetes spp.) | aphids, whiteflies, root-knot nematodes |
| Lemongrass (Cymbopogon citratus) | mosquitoes, gnats |
| Dill (Anethum graveolens) | aphids, spider mites, cabbage worms |
| Oregano (Origanum vulgare) | ants, aphids, mosquitoes |
| Thyme (Thymus vulgaris) | cabbage worms, whiteflies, mosquitoes |
| Geranium (Pelargonium spp.) | mosquitoes, leafhoppers |
| Bay laurel (Laurus nobilis) | pantry moths, weevils, cockroaches |
| Tansy (Tanacetum vulgare) | ants, flies, cucumber beetles |
