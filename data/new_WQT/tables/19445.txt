# State income tax return extensions  whether a federal extension is accepted (selected states)

| State | Accepts Federal Extension for State Filing? | State Extension Form Required? | Notes |
| --- | --- | --- | --- |
| California | Yes | No (if federal extension filed) | Payment may still be due by original deadline |
| New York | Yes | Sometimes | Separate state extension needed if no federal extension |
| Colorado | Yes | No | Automatic extension to file, not to pay |
| Alabama | No | Yes | Must file state extension form to get extra time |
| Oregon | Yes | No | Extension to file granted when federal extension is valid |
| South Carolina | Yes | No | Federal extension generally accepted for filing |
| Massachusetts | No | Yes | State extension form required regardless of federal |
| Arizona | Yes | No | Federal extension recognized for filing purpose |
