# Windows Shortcuts Using Alt+Letter (Not Alt-Number) for Diacritics

| Character | Name | Key Combination | Notes | Unicode (U+) |
| --- | --- | --- | --- | --- |
| ā | Latin small a with macron | Alt + A | Application-dependent shortcut | U+0101 |
| ī | Latin small i with macron | Alt + I | Often requires specific IME/layout | U+012B |
| ū | Latin small u with macron | Alt + U | Not the Windows Alt-number method | U+016B |
| ñ | Latin small n with tilde | Alt + N | Depends on keyboard layout | U+00F1 |
| ö | Latin small o with diaeresis | Alt + O | Not an Alt code (numeric keypad) | U+00F6 |
| ç | Latin small c with cedilla | Alt + C | May be menu accelerator in many apps | U+00E7 |
| ß | Latin small sharp s | Alt + S | Typically not a universal shortcut | U+00DF |
